# RAG-Based Question Answering System

This is a simple Retrieval-Augmented Generation (RAG) project built using FastAPI and LangChain.

## Overview
The system retrieves relevant documents from a vector database and generates answers using an LLM.

## Features
- Document retrieval using vector embeddings
- LLM-based response generation
- FastAPI backend for API endpoints

## Tech Stack
- Python
- FastAPI
- LangChain
- Google Generative AI (Gemini)
